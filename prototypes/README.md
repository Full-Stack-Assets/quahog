# Prototypes

Standalone, runnable prototypes — separate from the design docs and the engine projects.

## `Quahog3D.html`
A self-contained **offline 3D prototype** of Project QUAHOG (Three.js / WebGL,
in-canvas HUD, touch controls). Everything is inlined in the single file — no
build step and no network needed.

- **Run it:** open `Quahog3D.html` in any modern browser (desktop or phone).
- **Host it:** drop it on any static host (e.g. Vercel) to share a link.
